# Logo Usage Guidelines

## Core Principles

The memoria.uy logo is designed for **clarity, simplicity, and scalability**. Follow these rules to maintain brand consistency.

## ✅ Do

- **Use the provided SVG files** for best quality
- **Maintain aspect ratio** (logo is square, 1:1)
- **Use appropriate variant** (black-on-white or white-on-black) based on background
- **Ensure sufficient padding** (minimum 25% of logo height as clear space on all sides)
- **Scale proportionally** (lock aspect ratio when resizing)
- **Use at recommended minimum sizes**:
  - Web: 32px minimum
  - Print: 10mm minimum
  - Favicon: Use the provided 32x32 and 16x16 exports

## ❌ Don't

- **Don't rotate** the logo (always keep it upright)
- **Don't distort** (no stretching or squashing)
- **Don't add effects** (no shadows, glows, bevels, or gradients)
- **Don't change colors** (only black and white)
- **Don't add borders** or outlines (the grid pattern IS the logo)
- **Don't place on busy backgrounds** (ensure high contrast)
- **Don't round corners** (cells must remain sharp squares)
- **Don't modify the grid pattern** (use provided files as-is)

## Clear Space

Maintain a minimum clear space around the logo equal to **25% of the logo's height**:

```
┌─────────────────────────┐
│         ↕ 25%           │
│  ┌─────────────────┐    │
│  │                 │    │
│←→│   LOGO (100%)   │←→  │
│25│                 │25% │
│% │                 │    │
│  └─────────────────┘    │
│         ↕ 25%           │
└─────────────────────────┘
```

## Minimum Sizes

### Digital
- **Web/UI**: 32px × 32px minimum
- **Favicon**: Use provided 32px and 16px versions
- **Social media avatars**: 128px × 128px recommended
- **Extension icons**: Use provided 128px and 48px versions

### Print
- **Business cards**: 10mm × 10mm minimum
- **Posters/banners**: Scale as needed (no maximum)

## Background Pairing

| Logo Variant | Use On |
|--------------|---------|
| Black on White | Light backgrounds (#FFFFFF, #F5F5F5, light grays) |
| White on Black | Dark backgrounds (#000000, dark grays) |

## Incorrect Usage Examples

❌ Logo with gradient fill
❌ Logo with drop shadow
❌ Logo rotated 45°
❌ Logo with rounded corners
❌ Logo on low-contrast background (e.g., gray logo on gray background)
❌ Logo stretched horizontally
❌ Logo with added border/frame
❌ Logo with modified cell pattern

## File Formats

- **Web/Digital**: Use SVG for scalability
- **Favicons**: Use provided PNG exports (32px, 16px)
- **Social media**: Use PNG exports (128px, 256px, 512px)
- **Print**: Use SVG (vector) or high-res PNG (512px+)

## Accessibility

- Always ensure **WCAG AA contrast ratio** (4.5:1 minimum for text, 3:1 for large graphics)
- Use `alt="memoria.uy logo"` for accessibility when embedding in HTML
- Provide text alternative when logo is used as clickable link

## Questions?

If you're unsure about logo usage, refer to the source files in `src/logo/` or open an issue on the GitHub repository.
