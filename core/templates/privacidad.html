Política de Privacidad

Última actualización: 30 de diciembre de 2025

Memoria.uy es un proyecto académico y de interés social que respeta tu privacidad. Esta política explica qué información recopilamos, cómo la usamos y tus derechos.

Al usar memoria.uy (sitio web o extensión de navegador), aceptás las prácticas descritas en esta política.

1. Información que recopilamos

1.1 Información que vos proporcionás

  • • URLs de noticias: Las direcciones web de artículos que enviás para votar o archivar.
  • • Contenido HTML: Cuando usás la extensión de navegador, capturamos el contenido completo de la página para procesarlo (esto permite acceder a artículos detrás de paywalls).
  • • Votos: Tu opinión sobre cada noticia (buena/mala/neutral).
  • • Datos de cuenta (opcional): Si en el futuro activamos el registro de usuarios, podríamos recopilar email. Actualmente el registro está deshabilitado.

1.2 Información recopilada automáticamente

  • • Identificador de sesión: Un ID único generado aleatoriamente (UUID) para vincular tus votos de forma anónima. No contiene información personal.
  • • Dirección IP: Registrada temporalmente en logs del servidor para seguridad y depuración. No se vincula permanentemente a tus votos.
  • • Información técnica: Tipo de navegador, sistema operativo, timestamps de acceso (logs estándar de servidor web).

1.3 Información que NO recopilamos

  • • No rastreamos tu navegación fuera de memoria.uy
  • • No usamos cookies de terceros o trackers publicitarios
  • • No vinculamos tu identidad real a tus votos
  • • No vendemos ni compartimos datos con anunciantes

2. Cómo usamos la información

  • • Procesamiento de contenido: Usamos modelos de lenguaje (LLMs vía LiteLLM/OpenRouter) para extraer título, resumen, entidades y sentimiento de los artículos.
  • • Análisis agregado: Estudiamos patrones de votación de forma anónima para investigación sobre polarización mediática y consenso social.
  • • Funcionalidad del servicio: Vincular tus votos entre sesiones, mostrar filtros personalizados, sincronizar extensión con sitio web.
  • • Mejora del servicio: Detectar errores, optimizar rendimiento, prevenir abuso.

3. Compartir información con terceros

Compartimos información mínima con estos proveedores de servicios:

OpenRouter / Proveedores de LLM

Enviamos el contenido HTML de las noticias para procesamiento con inteligencia artificial. No enviamos tu ID de sesión ni información personal. Estos servicios tienen sus propias políticas de privacidad.

Hosting / Infraestructura

Usamos proveedores de hosting (Railway, potencialmente otros) que pueden tener acceso a logs del servidor y datos de infraestructura. Los datos se almacenan en bases de datos gestionadas por estos proveedores.

No vendemos tus datos. No compartimos información con anunciantes, brokers de datos, o redes sociales.

4. Retención de datos

  • • Votos y noticias: Se mantienen indefinidamente para valor histórico de investigación.
  • • Sesiones: Las sesiones de Django expiran según configuración estándar (típicamente 2 semanas de inactividad). El ID de sesión de la extensión persiste en tu navegador hasta que lo elimines.
  • • Logs del servidor: Se mantienen por máximo 30 días para depuración.
  • • HTML capturado: Se almacena para permitir re-procesamiento si es necesario.

5. Tus derechos

Dado que no recopilamos información personal identificable (los votos están vinculados a un ID de sesión anónimo), no podemos identificarte para ejercer derechos GDPR típicos (acceso, rectificación, eliminación).

• Eliminar tu sesión: Podés borrar cookies y localStorage de tu navegador en cualquier momento. Esto desvincula tus votos futuros de los anteriores.

• Dejar de usar el servicio: Podés desinstalar la extensión y dejar de visitar el sitio en cualquier momento.

• Contacto: Si tenés preocupaciones específicas sobre privacidad, contactanos a través de GitHub (ver sección de contacto).

6. Seguridad

Implementamos medidas de seguridad razonables para proteger los datos:

  • • Comunicación cifrada (HTTPS)
  • • Protección CSRF en formularios
  • • Validación de entrada para prevenir inyecciones
  • • Logs de acceso para detectar actividad sospechosa

Sin embargo, ningún sistema es 100% seguro. Usá el servicio bajo tu propio riesgo.

7. Extensión de navegador

La extensión de Chrome/Firefox de memoria.uy tiene permisos específicos:

activeTab

Accede al contenido de la página actual solo cuando hacés clic en el ícono de la extensión.

storage

Guarda tu ID de sesión localmente en tu navegador (chrome.storage.local). No se sincroniza a otros dispositivos.

scripting

Permite inyectar código solo en páginas de memoria.uy para sincronizar tu sesión.

host_permissions

Comunica únicamente con memoria.uy y localhost:8000 (desarrollo). No envía datos a otros sitios.

La extensión no rastrea tu navegación. El content script que se ejecuta en todas las URLs solo actúa cuando visitás memoria.uy para sincronizar sesiones.

8. Menores de edad

Memoria.uy no está dirigido a menores de 13 años. No recopilamos intencionalmente información de niños. Si descubrimos que hemos recopilado datos de un menor, los eliminaremos.

9. Cambios a esta política

Podemos actualizar esta política ocasionalmente. La fecha de "Última actualización" al inicio de la página refleja la versión más reciente. Cambios significativos serán comunicados en el sitio web o repositorio de GitHub.

10. Transparencia (Código abierto)

El código completo de memoria.uy es público y auditable:

github.com/raulsperoni/memoria

Podés verificar exactamente qué hace el código con tus datos. Aceptamos auditorías de seguridad y privacidad de la comunidad.
